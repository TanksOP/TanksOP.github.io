Dungeon Game C++

Key Info

  • Made using SDL Libarys
  • Language - C++
  • Team Size - Solo Project
  • Year - 2021/2022

This is a 2D dungeon game where you need to kill the goblins coming after you. The game uses the SDL libraries to help control the images and the sounds in the game. This was the first game I programed in C++ without using a game engine.
