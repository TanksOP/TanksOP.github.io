AR Breadboard

Key Info

  • Engine Used - Unity
  • Language - C#
  • Team Size - Solo Project
  • Year - 2023

This project was part of my final year project which I wrote my disertation on. Where I was creating an appliction to help students understand and learn how to create an electrical circuit on a breadboard. I decided that I would use Unity to create the game in as I was aware of the engien being capable to create AR visuals. The app works by poting your phone camera towards the QR code and it will display the instructions to create the circuit. The app only has one posible circuit to create but it could allow students to select more options in the future.

The project mainly uses two features of the engine outside of the AR capabilityes which were the canvas and the animation timelines. The canvas was used to display to the user what their current step was and allowed them to touch on the mobile device to move to the next step. The animation timeliens where used to move the electrical componants, to help show where they where meant to go.
