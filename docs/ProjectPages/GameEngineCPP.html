My Game Engine

Key Info

  • Made using OpenGL and SDL2
  • Language - C++
  • Team Size - Solo Project
  • Year - 2022

In this project I created a game engine. Then used it to create a wild west inspired scene.

Using the game engine you are able to: add models with textures, create a terrain, add in some fog, create a skybox or a sky dome, have a mini-map and create a HUD.

In my wild-west scene I also have a first person camera that can be moved around the scene, and added in the ability to change the fog density while the game is running.
